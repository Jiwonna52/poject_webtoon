# novelPlatform
## 간단한 설명
crud를 활용한 웹소설 연재 플랫폼입니다.
Spring boot 공부를 위해 무작정 시작한 프로젝트이지만, 애정을 가지고 계속해서 개발하고 있습니다.
적어도 10월까지는 주요 기능을 완성하는 것을 목표로 하고 있습니다.
## 개발 기간
23.05 ~ 현재
## 개발 인원
1명
## 개발환경
spring boot '2.7.11', java 11, JPA, gradle 7.6.1
## 데이터베이스
maria db
## 기타
html, css
